# Test Case Structure: Feedback Form

## Layout Test Cases

- Page background layout
- Form layout "Send us a message"
- Name field layout (in the form)
- Email field layout (in the form)
- Phone number field layout (in the form)
- Message field layout (in the form)
- Submit button layout

## Layout Test Cases (after sending / error states)

- Layout of the message about successful subscription
- Layout of the error message in the Name field
- Layout of the error message in the Email field
- Disappearing messages (error/success timeouts)

## Name Field Test Cases

- Checking the length of the Name field (valid lengths)
- Checking the entered characters in the Name field (valid characters)
- Checking the length of the Name field (invalid lengths)
- Checking the entered characters in the Name field (invalid characters)

## Email Field Test Cases

- Email field length check (valid lengths)
- Checking the characters entered in the Email field (valid characters)
- The Email field is required (empty validation)
- Email field length check (invalid lengths)
- Checking the entered characters in the Email field (invalid characters)

## Telephone Field Test Cases

- Checking the length of the Phone field (valid lengths)
- Checking the length of the Phone field (invalid lengths)
- Checking the entered characters in the Phone field (invalid characters)

## Message Field Test Cases

- Checking the length of the Message field (valid lengths)
- Checking the entered characters in the Message field (valid characters)
- Checking the length of the Message field (invalid lengths)
- Checking the entered characters in the Message field (invalid characters)
